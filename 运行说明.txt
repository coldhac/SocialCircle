Social Circle 运行说明

【环境要求】
- Java 11或更高版本
- JavaFX 11或更高版本
- Eclipse IDE

【运行步骤】
1. 打开Eclipse
2. File > Import > Existing Projects into Workspace
3. 选择SocialCircle文件夹
4. 右键Main.java > Run As > Java Application

【功能说明】

1. 发布动态
   - 点击"发布"按钮
   - 输入内容
   - 可选择图片
   - 点击"发布"完成

2. 浏览动态
   - 主界面显示所有用户的动态
   - 可以滚动查看

3. 点赞
   - 点击心形按钮点赞
   - 再次点击取消点赞

4. 评论
   - 点击💬按钮查看详情
   - 在详情页可以发表评论
   - 可以删除自己的评论

5. 搜索
   - 在搜索框输入关键词
   - 点击"搜索"按钮
   - 可以搜索内容和用户名

6. 排序
   - 选择"最新"按时间排序
   - 选择"最多赞"按点赞数排序

7. 用户主页
   - 点击用户名查看主页
   - 显示用户的所有动态

8. 删除
   - 只能删除自己的动态和评论
   - 点击"删除"按钮确认删除

【数据保存】
- 应用关闭时自动保存
- 下次启动自动加载
- 数据文件：data.ser

【默认用户】
- alice (当前用户)
- bob
- charlie

【注意事项】
1. 图片选择后只保存路径，不复制图片
2. 如果移动了图片，会显示加载失败
3. 首次运行会自动生成一些示例数据
