# Social Circle - 朋友圈应用

**CSYE 6200 - Final Project - Fall 2025**

## 项目简介

这是一个简单的类似朋友圈的社交应用，使用JavaFX开发。

## 功能

1. 发布动态（文字+图片）
2. 浏览动态列表
3. 点赞功能
4. 评论功能
5. 搜索功能
6. 排序功能（最新/最多赞）
7. 用户主页
8. 数据保存

## 涵盖的课程主题

1. **Class Definition** - User, Post, Comment类
2. **Inheritance** - Post和Comment继承SocialEntity
3. **Abstract Classes** - SocialEntity抽象类
4. **Interfaces** - Likeable接口
5. **Lists** - ArrayList存储帖子和评论
6. **Set/Maps** - HashSet存储点赞，HashMap存储用户
7. **Generics/Collections/Iterators** - 使用泛型集合和Iterator

## 运行方式

1. 在Eclipse中导入项目
2. 确保配置了JavaFX库
3. 运行Main.java
